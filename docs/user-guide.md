# SIEM User Guide

## Getting Started

### Installation

1. **System Requirements**
   - Minimum 16GB RAM
   - 4+ CPU cores
   - 500GB storage
   - Network connectivity

2. **Installation Steps**
   ```bash
   git clone [repository]
   cd siem
   ./install.sh
   ```
